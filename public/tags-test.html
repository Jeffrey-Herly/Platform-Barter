👤 User Journey Tests

Scenario 1: Create Item with Custom Tag
  1. Go to Create Item page
  2. Type tag name (e.g., "Laptop Gaming")
  3. Click "Create new: Laptop Gaming"
  4. Verify tag added to selection
  5. Fill rest of form and submit
  6. Verify item created with tag
Scenario 2: Reuse Existing Tag
  1. Create first item with tag "Elektronik"
  2. Create second item
  3. Type "Elek" in tag input
  4. Should show "Elektronik" (yellow badge)
  5. Click to add
  6. Verify tag linked to second item
Scenario 3: Permission Check
  1. User A creates custom tag "Photography"
  2. User B tries to delete User A's tag via API
  3. Should receive 403 Forbidden
  4. Verify tag still exists for User A
